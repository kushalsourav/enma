Buttons are used on forms, website homepages, dialog boxes, and toolbars. It is important to differentiate between buttons and links. Buttons are used when you want a user to act (submit, cancel, delete) and links are used to direct users to other pages

Examples of Solid Buttons

Solid Buttons are available in 3 formats - primary , secondary , tertiary.
you have to include these classes btn-primary btn-secondary btn-tertiary
you must include class btn and then its variants.

Example of Outline Button

Add these classes for outline button --- btn-primary-outline btn-secondary-outline btn-tertiary-outline

For icon include btn-icon icon class.
For link include btn-primary class or other classes inside < a> tag.
